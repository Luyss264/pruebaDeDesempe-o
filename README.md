# STUDENT MANAGEMENT SYSTEM

This is a system for managing information in an educational institution.

It allows the user to manage basic data such as adding, searching, updating, and deleting students from the system.

## Utilities

### Register a new students

To use it, simply select the option.
First, you'll be asked to enter a unique ID, then it will ask you to enter your name, age, grade, and status.

This information is automatically saved in the database

### Show the students

Selecting this option will show you all registered students. However, it will tell you there are no students yet if none are registered.

### Consult some student

By going to this option you can search for a specific student. To do this, it asks you to enter the student's name; once you do this, it will show you the student's information.

### Update a student's information

Searching by name also grants access to a student, but this time to edit their attributes.

Everything can be edited except the ID, as this is a unique and unrepeatable piece of data that has already been assigned, unless the student is deleted.

### Delete a student

To delete a student, you access the site by searching for the ID. Once the student is found, it will be deleted and will show you which student you deleted.

### Go out

Finally, if you choose the exit option, it will automatically kick you out; that will then be the end of the program.
